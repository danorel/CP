This program is meant for those, who'd like to learn more about an ancient problem, which could be solved in different ways.
Exist some kinds of algorithms, which could describe the moves, done by the horse, such as Euler's method, Warnsdorf's rule and the others.
This program is a perfect choice, because you can edit the board and the horse facilities , you can also analyse the position in any step. Also, you could change the language, find an awesome video in the last tab.
This program might be very beneficial for beginners, because in the next versions we will add some new problems! Thank you for using this program, good luck and have fun!
